## ğŸ“„ File-by-File Explanation

---

### ğŸ”§ ROOT FILES

| File | Purpose |
|------|---------|
| `.gitignore` | Tells Git which files NOT to upload (system files, temp files, etc.) |
| `netlify.toml` | Configuration for Netlify hosting - caching rules, security headers, URL redirects |
| `README.md` | Project documentation for developers - how to edit content, add images |
| `manifest.json` | PWA configuration - app name, icons, colors for "Add to Home Screen" |
| `sw.js` | Service Worker - caches files for offline access, faster loading |
| `index.html` | **Home Page** - hero section, stats, featured programs, updates preview |

---

### ğŸ“ PAGES FOLDER

| File | Purpose | Main Sections |
|------|---------|---------------|
| `about.html` | Club history & information | History, Mission, Vision, Facilities |
| `programs.html` | All training programs | Program cards with schedule, age, status |
| `events.html` | Annual events & festivals | Event details, dates, photo galleries |
| `library.html` | Library information | Status badge, book categories, opening notice |
| `updates.html` | All announcements | Filter buttons, update cards by category |
| `community.html` | Committee members | Executive committee, Sub-committees |
| `contact.html` | Contact information | Contact details, form, Google Maps |

---

### ğŸ“ DATA FOLDER (JSON)

These are your **content files** - edit these to change website content.

| File | Contains | Key Fields |
|------|----------|------------|
| `club-info.json` | Basic club info | name, tagline, history, mission, contact, social links, stats |
| `programs.json` | Array of programs | id, name, icon, image, schedule, ageGroup, status, enrollmentOpen |
| `events.json` | Array of events | id, name, date, location, description, gallery images, isFlagship |
| `library.json` | Library details | status (closed/open), categories, totalBooks, openingNotice |
| `updates.json` | Array of notices | date, title, content, category, isPinned, isActive |
| `community.json` | Committee data | executive members, subcommittees with members |

#### Quick Edit Examples:

**Add new update** â†’ Edit `updates.json`, add object at TOP of array

**Activate library** â†’ Change `"status": "closed"` to `"status": "open"` in `library.json`

**Add new program** â†’ Add new object to `programs.json` array

### ğŸ“ CSS FOLDER

| File | Purpose | What It Contains |
|------|---------|------------------|
| `reset.css` | Browser normalization | Removes default margins, consistent box-sizing |
| `variables.css` | Design tokens | Colors, fonts, spacing, shadows, z-index (change here to rebrand) |
| `typography.css` | Text styling | Headings, paragraphs, links, Bengali font support |
| `layout.css` | Page structure | Container, grid, flexbox, section spacing, hero layout |
| `components.css` | UI elements | Buttons, cards, badges, forms, modals, alerts |
| `navigation.css` | Navigation | Header, mobile menu, footer, back-to-top button |
| `utilities.css` | Helper classes | Margins, padding, colors, animations, visibility |
| `pages.css` | Page-specific | Home hero, about sections, program cards, event cards |
